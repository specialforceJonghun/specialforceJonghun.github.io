## 저장하기
### csv 파일로 저장
ex)1,2,3,4,5,6,7,8


```python
df.to_csv('score.csv', encoding = 'utf-8-sig')
```


```python
df.to_csv('score.csv', encoding = 'utf-8-sig', index = False)
# 인덱스값 빼는거.
```

### 텍스트(.txt)파일로 저장


```python
df.to_csv('score.txt', sep ='\t')  # tab 으로 구분됨 즉 띄워서 됨
```

### 엑셀파일로 저장


```python
df.to_excel('score.xlsx')
```

## 열기
### csv파일 열기

df = pd.read_csv('score.csv')
